# ApexPlanet Internship: Product Manager

This project is a simple E-commerce Product Management system built for the ApexPlanet Web Development Internship.

- **Intern:** [Your Name Here]
- **Technologies:** PHP, MySQL, Bootstrap
- **Features (Tasks 1-5):**
  - **Task 1:** Git repository and local XAMPP environment setup.
  - **Task 2:** Full CRUD (Create, Read, Update, Delete) functionality for products.
  - **Task 2:** User Authentication (Register, Login, Logout) with session management.
  - **Task 3:** Bootstrap UI, product search, and pagination.
  - **Task 4:** Full security implementation:
    - Prepared Statements (MySQLi) to prevent SQL Injection.
    - Server-side and client-side form validation.
    - Role-Based Access Control ('admin' vs. 'user').
  - **Task 5:** All features are fully integrated and tested.
